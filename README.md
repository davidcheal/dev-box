# DevBox Builder

This script will install all the applications to create a new devloper environment. I typically run this in a VMWare Virtual Machine, though it could be run on a physical machine

- PHP
- Ruby
- NodeJS / npm
- xcode (Mac Only)

- Anaconda AI/ML IDE

- Docker
